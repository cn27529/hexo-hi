功能 : 設定文字和邊緣距離 ( 可設單位屬性 : 點pt、英寸in、公分cm、像素px、百分比% )
語法 : { MARGIN : AUTO︱( length )︱( percentage ) }

這個性質一次設定上、下、左、右邊界與區塊外之元件間格距離。使用時應依序指定上、右、下、左，如margin:1px 2px 3px 4px;。亦可一次指定四邊，如margin:2px，則全部均設2px。

1.
margin:1px 2px 3px 4px;

2.
margin-top:1px;
margin-right:2px;
